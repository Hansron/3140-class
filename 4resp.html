This class is designed to give you experience in a number of the online tools available to journalists today. I've created links to examples of each of the tools we'll use. This page also serves as the example of the last skill, basic Web design. Click on the links above to take a closer look.

One other element I've added to the CSS of this page is responsiveness. Instead of using absolute values for DIV sizes and even image sizes, I've used percentages.

I've also added a Twitter widget to provide a really basic example of how to install and use Javascript.

By the way, I'm Hans K. Meyer, a professor at the E.W. Scripps School of Journalism at Ohio University . I've been at OU for nearly 10 years. It's hard to believe time has gone by that fast.
